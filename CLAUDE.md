# MeuSalão — Contexto do Projeto

## O que é

Marketplace digital de serviços de beleza com foco em descoberta local e agendamento direto. Conecta clientes a salões próximos, centralizando oferta, agenda e avaliações. Mobile-first.

## Estratégia de produto

### Visão de longo prazo

O MeuSalão começa como plataforma de visibilidade e agendamento, mas a prioridade é **gradualmente se tornar o sistema central de uso do salão**. A cada nova funcionalidade liberada, o salão tem menos motivos para manter sistemas paralelos.

### Jornada de adoção para salões

1. **Entrada:** Salão se cadastra na plataforma para ganhar visibilidade e receber agendamentos de clientes novos
2. **Importação:** Importa planilhas com base de clientes, serviços e preços do sistema atual — migração sem retrabalho
3. **Uso paralelo:** Mantém o sistema antigo para gestão interna enquanto usa o MeuSalão para captação e agendamento online
4. **Migração progressiva:** Conforme novas funcionalidades são liberadas no MeuSalão, vai transferindo processos do sistema antigo para a plataforma
5. **Centralização:** O MeuSalão se torna o sistema principal do salão, eliminando a necessidade do sistema anterior

### Princípios de integração

- **Aceitamos dados externos:** a plataforma permite importação de dados de outros sistemas (planilhas, CSV) para facilitar a entrada de salões que já têm base de clientes e catálogo cadastrados em outro lugar
- **Migração sem retrabalho:** o salão não precisa redigitar dados — importa e começa a usar
- **Funcionalidades liberadas gradualmente:** cada nova feature (gestão de agenda completa, relatórios, controle de equipe) é um passo a mais para o salão centralizar tudo no MeuSalão
